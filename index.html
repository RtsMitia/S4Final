Système de Gestion Financière

Comment utiliser l'application :

🔍 Rechercher un client : Utilisez la barre de recherche dans le menu de gauche

📋 Consulter les détails : Cliquez sur un client pour voir ses informations et prêts

💰 Créer un prêt : Depuis la fiche client, cliquez sur "Faire un prêt"

📊 Gérer les paramètres : Utilisez les menus pour configurer les établissements et types de prêts
